# 慈濟海報資料庫後台 — 各頁面設計提示詞（完整版）

> **統一規範**
> - 頁面背景 `#F8F9FA`
> - 所有卡片容器統一使用 **card-box** 樣式：`position: relative; margin: 8px; border-radius: 16px; padding: 8px; background-color: white; box-shadow: 0 0 10px rgba(128,128,128,0.63);`
> - 深藍 `#003366` 僅用於按鈕、標題、icon、邊框、選中狀態
> - 字體 Noto Sans TC
> - 元件庫 shadcn/ui + Tailwind CSS 4
> - 所有 Label 深藍色 `#003366`，必填星號紅色 `#DC3545`
> - Input 灰邊框，聚焦時深藍邊框 + 淺藍光暈 `#003366/10%`
> - 唯讀欄位統一灰底 `#F0F0F0`
> - 狀態 Badge 色系：草稿灰底灰字、待審核淺黃底棕字、已上架淺綠底綠字、已退回淺紅底紅字、審核中淺藍底藍字、待結案淺紫底紫字
