# 12 主題 banner 生圖 prompt

參考來源：`/Users/webit/Downloads/banner-lg.jpg`（慈濟典型 banner 風格）

## 通用視覺風格鎖定（每段都複製這段）

> **背景**：深海藍漸層（左上 `#1a2b6e` 到右下 `#0a1640`），上面疊白色細曲線流線（像等高線、流暢有韻律），點綴金色虛線光帶（像 LED 燈串）蜿蜒穿過，右上角隱約透出地球意象。
>
> **人物**：慈濟志工，穿藏青色短袖制服配白色立領（女性藍色長裙、男性藍色長褲），戴白色帽或藍色帽，神情虔敬專注。臉部清晰、構圖以人物中景為主、其他元素圍繞。
>
> **質感**：照片級寫實、明亮但帶莊嚴感、整體偏冷色但人物臉部與光帶為暖色點綴形成對比。
>
> **構圖**：橫式 banner、寬畫面、主題人物在右半或中央偏右、左半留空背景給標題字。
>
> **標題文字**：中央偏左、白色或淡金色、繁體中文書法或現代黑體粗字、字級突出但不喧賓奪主、可加細微光暈或陰影。
>
> **長寬比**：16:9（一般使用）或 21:9（hero banner 加寬版）。
>
> **避免**：卡通風、低彩度、過度模糊、文字錯字、變形手指。

## 使用建議

### 哪個工具用哪個版本

| 工具 | 推薦版本 | 備註 |
|---|---|---|
| ChatGPT 4o / GPT-Image | **中文敘述版**（每段第一個 blockquote） | 中文 prompt 表現好；可一次貼一段、產 2-4 張選 |
| Sora / Imagen / Recraft | **中文敘述版** | 同上 |
| Midjourney | **MJ 版**（英文 + 參數） | 注意中文字 MJ 6 渲染不穩，建議文字事後 Photoshop / Figma 疊上 |
| Stable Diffusion (Flux / SDXL) | **中文敘述版**翻英譯 + LoRA 控字 | 中文字一樣建議事後疊 |

### 字體一致性

12 張都用同一字體，最好是繁體粗體（思源宋體 Bold / 文鼎風行體 / 標楷體加粗）或書法體。
建議生圖時讓 AI 留出文字位置（用「左半留空給標題字」這句），事後用 Figma / Photoshop 統一疊字 — 比讓 AI 渲染中文更穩。

### 12 張一致性技巧

- 12 段 prompt 一次跑完不要中斷（同個 session 跑能保留風格 cache）
- 同一個 seed（如果工具支援）只跑一次當基準，其他用同 seed 但換 prompt
- 跑完看一遍，色溫 / 構圖 / 人物 dress code 不一致的單獨重生
- 一定保留來源檔，未來主題 rename 也只是換標題字、底圖不用重生

### 整合進專案

生好後：
1. 圖檔放 `public/themes/` 下，命名 `<code>.jpg`（例如 `charity.jpg`）
2. 跑 SQL 更新 `vocabulary_themes.cover_image`：
   ```sql
   UPDATE vocabulary_themes SET cover_image = '/themes/' || code || '.jpg';
   ```
3. 前端 ThemeEditModal 已支援 cover_image 編輯，admin 也可手動單張更新
